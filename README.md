# Polo Playas Storefront

GitHub Pages-ready storefront.

## Publish
Upload every item in this folder to the root of the GitHub repository. Keep the `assets` folder intact.

## Included
- 11 product colorways
- Sizes XS through XXL
- Product search, filtering, sorting and details
- Shopping cart and demo checkout
- Logo and full-quality product imagery
- Demo admin dashboard

Prices and inventory values are placeholders until confirmed by the store owner. Real payments require Shopify, Stripe, Square or another secure commerce backend.
